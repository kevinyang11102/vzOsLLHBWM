# 餐馆评论系统 python1267

## 项目概述

餐馆评论系统是基于Python、MySQL和Django框架开发的，旨在提供餐馆信息管理与用户评论功能。系统通过后端技术栈和前端技术栈的结合，实现了一套完整的餐馆信息展示和用户互动平台。

## 技术栈

### 开发环境
- 操作系统: Windows 10/11
- 开发工具: PyCharm 2023.1
- 数据库: MySQL 8.0
- Python版本: Python 3.8+

### 后端技术栈
- Web框架: Django 4.0
- 数据库ORM: Django ORM
- API开发: Django REST framework
- 权限管理: Django Auth

### 前端技术栈
- HTML5
- CSS3
- JavaScript
- Bootstrap 4
- jQuery

### 数据库工具
- MySQL Workbench
- Navicat Premium

## 功能模块
- 餐馆信息管理
- 用户评论发布与管理
- 用户权限认证
- 数据库信息维护

## 系统角色
- 系统管理员：负责餐馆信息的审核和管理。
- 普通用户：可以发布评论，查看餐馆信息。

## 运行环境
- 服务器：兼容Django 4.0的服务器环境
- 数据库：MySQL 8.0
- 客户端：现代浏览器

## 部署步骤
1. 配置好Python 3.8+环境。
2. 安装必要的Python库，如Django 4.0等。
3. 使用MySQL Workbench或Navicat Premium创建数据库。
4. 运行Django项目迁移数据库命令。
5. 启动项目服务。

## 核心亮点
- 基于Django REST framework，支持API开发，易于扩展。
- 使用Django ORM，方便进行数据库操作。
- 前端采用响应式设计，适应多种设备访问。
- 集成Django Auth，实现用户权限的精细化管理。
